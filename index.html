¿Que es HTML?

HTML es un lenguaje de marcado que utiliza etiquetas para estructurar el contenido de una página web, como texto, imágenes, enlaces y otros elementos.
¿Cómo funciona?
Las etiquetas HTML, como

para un párrafo o para una imagen, indican a los navegadores web cómo mostrar el contenido.

¿Que es CSS?

CSS (Cascading Style Sheets)

es un lenguaje de programación que se utiliza para definir el estilo de las páginas web. Se le conoce como "Hojas de Estilo en Cascada" porque las características se aplican de arriba abajo.

¿que es JavaScript?

JavaScript (JS) Es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, ampliamente utilizado para crear páginas web interactivas y dinámicas, así como aplicaciones web y de escritorio.
